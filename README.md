# fundamental-programming

# To run Go

```shell
go run main.go
```

# To run Java

- compile

```shell
javac main.java
```

- excute

```shell
java main
```

# To run JavaScript

```shell
node main.js
```

# To run Clojure

```shell
java -cp clojure-1.6.0.jar clojure.main
```

# To run Swift

```shell
swift main.swift
```

# To run C++

- install mingw-w64

```
brew install mingw-w64
```

- run c++ file

```shell
make cpp-run
```
